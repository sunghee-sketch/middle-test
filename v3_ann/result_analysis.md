## 2. 모델 구조

```
입력(2431) → Linear(512) → BN → ReLU → Dropout(0.3)
           → Linear(256) → BN → ReLU → Dropout(0.3)
           → Linear(128) → BN → ReLU → Dropout(0.2)
           → Linear(1)
```

- 손실: MSE (sample weight 반영, 고MP 상위 10%에 가중치 3.0)
- 옵티마이저: Adam(lr=1e-3, weight_decay=1e-4)
- 스케줄러: ReduceLROnPlateau(patience=10, factor=0.5)
- 조기 종료: patience=20
- 타깃 변환: `log1p(MP)` 학습 → 예측 후 `expm1()` 역변환
- 안정화: Gradient clipping(max_norm=5.0), 피처 NaN/Inf 제거, 스케일 후 `[-10, 10]` 클리핑

## 6. 고찰

### 핵심 성과: CV·Test 모두 상승

ANN 도입으로 **CV R² +0.172, Test R² +0.099**의 동반 향상을 얻었습니다. 특히 Test MAE가 53.06 K → 46.35 K로 약 **6.7 K 개선**되어, 녹는점 예측 실용 정확도가 의미 있게 높아졌습니다.

트리 모델(XGBoost)이 포착하지 못했던 **고차원 비선형 상호작용**을 MLP의 은닉층이 학습했기 때문으로 판단됩니다. Morgan + MACCS + RDKit 기술자(2431차원)의 조합에서, 트리는 이산 분기로 근사하는 반면 ANN은 연속적 가중치 합성으로 복잡한 함수를 직접 근사할 수 있습니다.

---

### 새로운 문제: CV-Test 격차 확대

| | base | v3_ann |
|---|------|--------|
| CV-Test 격차 | 0.052 | **0.124** |
| CV 편차 | ±0.079 | **±0.120** |

base에서 해소됐던 과적합 경향이 v3_ann에서 다시 커졌습니다.

- **Fold 2 (R²=0.483)** 와 **Fold 5 (R²=0.815)** 사이의 격차가 0.33으로 매우 큽니다 — 특정 분할에 대한 민감도가 존재합니다.
- Test R²(0.58)가 CV 최고 Fold(0.82)보다 CV 최저 Fold(0.48)에 가깝습니다 — 외삽 영역에서 성능 저하.

원인 추정:
1. **샘플 수 부족**: 2117개는 2431차원 입력에 비해 적어 과적합 취약.
2. **고MP 영역 희소성**: 상위 10% 가중치(3.0) 부여로 완화했으나, 고MP 분포가 편중된 fold에서 편차 확대.
3. **Dropout 강도**: 0.3 수준이 이 데이터 규모엔 다소 약할 수 있음.

---

### 트레이드오프 해석

base는 "안정적이지만 성능 한계가 낮은 모델", v3_ann은 "평균 성능은 높으나 분할 의존성이 큰 모델"입니다. 실용 관점에서는 MAE가 더 낮은 v3_ann이 유리하지만, 신뢰 구간이 넓어 극단 샘플에 대한 예측은 조심스럽게 해석해야 합니다.

## 7. 향후 개선 방향

1. **Dropout 0.3 → 0.5 또는 Weight Decay 강화** — CV 편차와 CV-Test 격차 축소
2. **앙상블**: 서로 다른 seed의 MLP 5개 평균 → 분할 의존성 완화
3. **하이브리드**: v2 트리 앙상블과 v3_ann의 평균/스태킹 → 양쪽 강점 결합
4. **데이터 증강**: SMILES canonical/atom ordering 변형으로 유효 샘플 확장
5. **고MP 샘플 SMOTE-R** 또는 가중치 상향(3.0 → 5.0)으로 Fold 2 유형의 실패 완화
6. **v3_gnn 비교**: GNN이 비선형 학습에서 더 나은 일반화를 보일 가능성
